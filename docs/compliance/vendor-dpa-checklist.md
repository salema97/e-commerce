# Data Processing Agreements (DPAs)

Plantilla de seguimiento para cumplimiento GDPR. Marca cada proveedor cuando el DPA esté firmado o aceptado en su panel.

| Proveedor | Servicio | DPA / Privacy | Estado |
|-----------|----------|---------------|--------|
| Auth (JWT nativo) | Autenticación | Política interna + hosting | En producción |
| Stripe | Pagos | Dashboard → Legal | Pendiente |
| Vercel | Hosting web | Settings → Legal | Pendiente |
| Cloudflare | CDN / R2 | Dashboard → Privacy | Pendiente |
| Evolution API | WhatsApp (self-hosted) | Responsabilidad propia | Pendiente |
| Resend / Loops | Email | Panel proveedor | Pendiente |
| PostHog | Analítica | PostHog DPA | Pendiente |
| Sentry | Errores | Sentry DPA | Pendiente |
| OneSignal / Expo | Push | Panel proveedor | Pendiente |

**Nota:** Este archivo es un checklist operativo; la revisión legal final es responsabilidad del negocio.
